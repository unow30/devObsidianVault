# 웹 인터페이스 가이드라인

파일이 웹 인터페이스 가이드라인(Web Interface Guidelines)을 준수하는지 검토합니다.

## 작동 방식

1. 아래 소스 URL에서 최신 가이드라인을 가져옵니다.  
2. 지정된 파일(또는 사용자에게 파일/패턴을 입력받음)을 읽습니다.  
3. 가져온 가이드라인의 모든 규칙과 비교해 검사합니다.  
4. 결과를 `file:line` 형식의 간결한 출력으로 제공합니다.

## 가이드라인 소스

리뷰를 시작할 때마다 최신 가이드라인을 불러옵니다:

```
https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md
```

WebFetch를 사용해 최신 규칙을 가져옵니다.  
가져온 내용에는 모든 규칙과 출력 형식 지침이 포함되어 있습니다.

## 사용 방법

사용자가 파일 또는 패턴 인자를 제공한 경우:  
1. 위의 소스 URL에서 가이드라인을 가져옵니다.  
2. 지정된 파일을 읽습니다.  
3. 가져온 가이드라인의 모든 규칙을 적용합니다.  
4. 가이드라인에 지정된 형식으로 결과를 출력합니다.

파일이 지정되지 않은 경우, 어떤 파일을 검토할지 사용자에게 문의합니다.
