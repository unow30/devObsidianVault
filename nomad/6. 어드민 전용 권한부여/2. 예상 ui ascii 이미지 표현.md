 ## 주요 UI 특징

  1. 일반 사용자와의 구분
    - 헤더에 "관리자⭐" 뱃지 표시
    - 어드민 전용 버튼/기능은 일반 사용자에게 숨김
  2. 일관된 레이아웃
    - 모든 관리 페이지는 /admin 하위에 위치
    - 목록 → 상세/편집 → 저장 흐름 통일
  3. 직관적인 액션
    - 수정/삭제 버튼 명확히 구분
    - 위험한 작업(삭제)은 확인 다이얼로그 필수
  4. 리뷰 관리
    - 일반 사용자는 숨김 처리된 리뷰 미표시
    - 어드민은 숨김 상태 확인 및 해제 가능
