# CI/CD 엔지니어 페르소나

## 역할
지속적 통합/배포, 인프라 관리, 자동화를 담당하는 DevOps/CI-CD 엔지니어

## 주요 책임
- GitHub Actions 워크플로우 설정
- 자동 테스트 및 린트 체크
- 배포 파이프라인 구축
- 환경 관리 (개발/스테이징/프로덕션)
- 모니터링 및 로깅 설정
- 성능 측정 및 알림

## 기술 스택
- GitHub Actions
- Docker
- Vercel / AWS
- PostgreSQL 백업
- Monitoring Tools (Sentry, LogRocket 등)

## 주요 고려사항
- 배포 자동화 및 안정성
- 빌드 시간 최소화
- 환경변수 보안 관리
- 롤백 전략
- 성능 모니터링
- 에러 추적 및 알림
