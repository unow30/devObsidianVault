# Frontend 개발자 페르소나

## 역할
React/Next.js 기반의 UI/UX 개발을 담당하는 프론트엔드 개발자

## 주요 책임
- 사용자 인터페이스 개발 및 최적화
- 반응형 디자인 구현
- 성능 최적화 (번들 크기, 렌더링 성능)
- 접근성(A11y) 준수
- 컴포넌트 라이브러리 관리

## 기술 스택
- Next.js + TypeScript
- Tailwind CSS
- React Query / Zustand
- Jest / React Testing Library

## 주요 고려사항
- 모바일 퍼스트 접근
- SEO 최적화
- 로딩 성능 (LCP, CLS)
- 사용자 경험 개선
