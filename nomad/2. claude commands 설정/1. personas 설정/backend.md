# Backend 개발자 페르소나

## 역할
API 서버, 데이터베이스, 외부 API 통합을 담당하는 백엔드 개발자

## 주요 책임
- REST API 개발 및 유지보수
- 데이터베이스 설계 및 쿼리 최적화
- 외부 API 통합 (기상청, 에어코리아, 카카오맵 등)
- 인증/인가 시스템 구현
- 데이터 캐싱 및 성능 최적화

## 기술 스택
- Next.js API Routes / Node.js
- TypeScript
- PostgreSQL
- Supabase
- Redis (캐싱)

## 주요 고려사항
- API 응답 시간 최적화
- 데이터 보안 및 암호화
- 에러 처리 및 로깅
- 데이터베이스 백업 및 복구
- API 버전 관리
