| 패턴                  | 주요 기능    | 사용 사례                  |
| ------------------- | -------- | ---------------------- |
| **Sidecar**         | 보조 기능 수행 | 로그 수집기, 프록시, 설정 관리     |
| **Adapter**         | 데이터 변환   | 로그 변환, 메트릭 변환          |
| **Ambassador**      | 프록시 역할   | API Gateway, DB 연결 프록시 |
| **Init Container**  | 선행 작업 실행 | 설정 파일 다운로드, DB 마이그레이션  |
| **Leader Election** | 리더 노드 선출 | 데이터 동기화, 분산 태스크 관리     |
